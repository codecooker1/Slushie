About

What is it?

This is just a super simple anonymous note/quote sharing app. No bloat, no selling your soul/data. Simply drop your thoughts or fav quote without any fuss, read what others have left behind, and bounce. It really is that easy.

“Saying you don't need privacy because you have nothing to hide is like saying you don't need freedom of speech because you have nothing to say.”

— Edward Snowden
Tech Stack

The absolute core part of it is built in python using Flask. The server side pages are generated using Jinja2 template engine and everything is stored in a single SQLite DB.

A funny thing I just realized: I have not written a single line of JS. Also, I only wrote a couple of lines of CSS thanks to PicoCSS handling all the heavy lifting. Moreover, writing HTML without the ‘<div>’ hell of bloated JS frameworks is quite relaxing tbh.

I can't really put my finger on exactly why but it feels like wayy less PITA to develop web stuff when I don't have to wrestle with the modern JS ecosystem??

“JavaScript makes me want to flip the table and say ‘Fuck this shit’, but I can never be sure what ‘this’ refers to.”

— @oscherler
Final Thoughts

My goal with this project was to KISS, like as much as possible. If a feature meant adding unnecessary complexity, it got the axe. Regardless of whether that might have made it more usable or not.

I don't know what else to write here. It does what it says on the tin. Oh, one more thing before I finish, the code is fully open-source on GitHub under GPLv3 license. So, feel free to make contributions if you spot a bug or wanna add some missing feature or star the project like.

Hope your life be peaceful and your contribution graph stay green!
